# study-languages

Daily practice repository for Python, SQL, and JavaScript.  
The purpose of this repository is to build strong programming fundamentals
and demonstrate consistent daily coding practice.

Python, SQL, JavaScript 기초를 매일 학습하고 기록하기 위한 저장소입니다.  
비전공자이지만 개발과 데이터 분석에 진지하게 접근하고 있으며,  
꾸준한 코딩 습관과 기초 실력을 쌓아가는 과정을 명확히 보여주는 것을 목표로 합니다.

---

## Why this repository exists

I created this repository to prove long-term consistency rather than short-term results.  
By committing small but meaningful code every day, I aim to build a solid foundation
that supports data science, machine learning, and product development.

이 저장소는 단기간의 성과보다 **지속적인 학습과 기록**을 증명하기 위해 만들었습니다.  
매일 적은 양이라도 의미 있는 코드를 커밋하며,  
데이터 분석·머신러닝·서비스 개발로 확장 가능한 **기초 역량**을 다지는 데 집중합니다.

## Daily routine (rule)

### English
- Minimum requirement: **one meaningful commit per day**
- A meaningful commit can be:
  - 10+ lines of code
  - small refactoring with comments
  - short documentation updates (README or notes)

### 한국어
- 하루 최소 **1회 의미 있는 커밋**
- 의미 있는 커밋의 기준:
  - 10줄 이상의 코드 추가
  - 주석을 포함한 간단한 리팩토링
  - README 또는 노트 정리

Completion is not required.  
Consistency and documentation are the priority.

완성도를 목표로 하지 않습니다.  
**지속성과 기록**을 가장 중요한 기준으로 삼습니다.
